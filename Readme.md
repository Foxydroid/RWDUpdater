## Rain World Devourment Updater

Easily install or update the **Devourment-mod** for Rain World in one command!

### Quick Start

#### Just download and run [RWDUpdater.bat](https://raw.githubusercontent.com/Foxydroid/RWDUpdater/refs/heads/main/RWDUpdater.bat)

or

#### Run The following command in PowerShell:

```ps1
iwr -useb "https://raw.githubusercontent.com/Foxydroid/RWDUpdater/refs/heads/main/RWDUpdater.ps1" | iex
```

- You **don't need to download any files manually**.
- The script auto-detects your Steam Rain World mods folder, or prompts you to select it.
- The window will close after finishing – check your Rain World mods folder for the update.

***

### What does it do?

1. **Detects Rain World mods folder.**
2. **Downloads the latest mod version** (via Git or Direct download).
3. **Compares with your installed files** using hashes.
4. **Installs/updates** the mod if needed.
5. **Cleans up** all temporary files.

***

### Requirements

- Windows (PowerShell 5.1+ or PowerShell Core recommended)
- Internet connection

**Optional:**  
- [Git for Windows](https://git-scm.com/) — for fastest updates and to see last commit date. If Git is not installed, ZIP download is used automatically.
